## history-enabled-calc


# Simple Calculator with History

A beginner Python calculator that performs basic arithmetic and saves calculation history to a file.

## Features
- Add, Subtract, Multiply, Divide
- Saves history to `history.txt`
- View history (`history` command)
- Clear history (`clear` command)

#  Example session

Welcome to a Simple Calculator
Type the equation: 5 + 3
Result: 8

Type the equation: 10 * 2
Result: 20

Type the equation: history
Result: 
10 * 2 = 20
5 + 3 = 8

Type the equation: exit
Result: 
Goodbye
